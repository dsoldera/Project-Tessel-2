# Project-Tessel-2
Project to Create Node Plugins to run on Tessel 2

## Installing the Tessel 2 Node Protocol
1. Run `npm install -g t2-cli `
2. Conect the Tessel on the computer using an USB
3. Run to check the connection `t2 list`
